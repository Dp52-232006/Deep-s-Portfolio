My Projects

By using the skills I have learned in the NexTech Internship I created interactive games and websites. These are Projects that I created during my time as a NexTech Catapult. Click on each one to view the project!


Air Pollution: Website

Using html and css only, I made a website that had information on air pollution and how it can be addressed locally in our community. Air Pollution is a common issue we currently face today and by creating this website it enlightens the user to how much of an impact air pollution has on today's society and what things they themselves can do to contribute to the goal or outcome.

Choose Your Own Adventure Game

Using html and css as well as Javascript, I made a game that is interactive with the user on what fate they decide. It is themed Amazon Rainforest invovling being lost in the Amazon Rainforest and choosing how you will go about escaping or your fate.

Which Emotion Are you? : Inside Out Edition

Using HTML, CSS, and Javascript, I made a quiz game that tells you which emotion you are based on the answers you choose to the series of questions. Figure out which emotion character you are by playing the game.
